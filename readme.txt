Requirements:

	- Android device
	- Version 4.0 (or better)


----------------------

To install on device (recommended):
	
	- Download app here: goo.gl/yyT19
	- OR, put MoneyManagement-1.1.2.apk file on device and manually install.




----------------------

To run the code locally:
	- Install Android SDK: http://developer.android.com/sdk/index.html
	- Update the SDK and install all versions of Android.

	- Create new project using existing code (from zip).
	- Run code using the Android emulator.
